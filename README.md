# CatchUp

CatchUp is a planning and focus assistant that helps users break work into daily targets, stay focused, and catch up when they fall behind.

## Problem
Static study or work plans often break when users miss days. Most tools either track tasks or offer timers, but they do not adapt when real life interrupts the plan.

## Solution
CatchUp helps users:
- break work into daily targets
- manage multiple parallel plans
- use focus sessions for structured work
- recover when they fall behind
- reduce overwhelm with a gentle restart flow

## Core Features
- Landing page and multi-plan dashboard
- Create and manage multiple plans
- Daily target calculation
- Focus session support
- Adaptive rescheduling
- Overwhelm recovery flow
